# Swarm Recruit Command

Execute a complete AI-powered recruitment workflow using swarm intelligence.

## Usage
```
/project:swarm-recruit "Job Title or Requirements"
```

## Workflow Steps

1. **Initialize Recruitment Swarm**
   - Set up hierarchical topology for organized execution
   - Spawn specialized recruitment agents
   - Configure memory for cross-agent coordination

2. **Analyze Requirements**
   - Parse job description with NLP
   - Extract key skills and qualifications
   - Identify cultural fit indicators
   - Generate ideal candidate profile

3. **Multi-Platform Sourcing**
   - Search LinkedIn, GitHub, Indeed simultaneously
   - Use AI to generate optimal search queries
   - Apply platform-specific optimizations
   - Aggregate candidate pool

4. **AI-Powered Screening**
   - Score candidates against requirements
   - Apply predictive success models
   - Rank by multiple criteria
   - Generate detailed match explanations

5. **Engagement Optimization**
   - Create personalized outreach messages
   - Optimize send timing
   - Track response rates
   - A/B test messaging strategies

## Required Tools
- `mcp__claude-flow__swarm_init`
- `mcp__claude-flow__agent_spawn`
- `mcp__claude-flow__task_orchestrate`
- `mcp__ruv-swarm__daa_agent_create`

## Options
- `--max-candidates`: Maximum candidates to process (default: 50)
- `--platforms`: Specific platforms to search
- `--location`: Geographic constraints
- `--remote`: Include remote candidates
- `--diversity`: Apply diversity optimization
- `--export`: Export results format (csv, json, pdf)
