# AI Optimize Command

Optimize your recruitment process using AI-powered analysis and improvements.

## Usage
```
/project:ai-optimize --target [process|job|campaign|workflow]
```

## Optimization Targets

### Process Optimization
- Analyze current recruitment workflow
- Identify bottlenecks and inefficiencies
- Suggest AI-powered improvements
- Implement automation opportunities

### Job Description Optimization
- Enhance job postings for better reach
- Optimize for SEO and job boards
- Improve clarity and appeal
- Add diversity-friendly language

### Campaign Optimization
- Improve outreach effectiveness
- Optimize messaging and timing
- Enhance response rates
- A/B test strategies

### Workflow Optimization
- Streamline multi-step processes
- Reduce time-to-hire
- Improve candidate experience
- Automate repetitive tasks

## Workflow Steps

1. **Current State Analysis**
   - Gather existing metrics
   - Identify pain points
   - Map current process
   - Baseline performance

2. **AI-Powered Assessment**
   - Apply machine learning models
   - Analyze patterns in data
   - Compare to best practices
   - Generate insights

3. **Optimization Recommendations**
   - Prioritized improvement list
   - Implementation roadmap
   - Expected impact analysis
   - Resource requirements

4. **Implementation Support**
   - Step-by-step guidance
   - Automation scripts
   - Template generation
   - Progress tracking

## Required Tools
- `mcp__claude-flow__bottleneck_analyze`
- `mcp__claude-flow__performance_report`
- `mcp__claude-flow__neural_patterns`
- `mcp__claude-flow__daa_optimization`

## Options
- `--metric`: Specific metric to optimize (time, cost, quality)
- `--baseline`: Include baseline comparison
- `--simulate`: Run simulation before implementing
- `--report`: Generate detailed report
